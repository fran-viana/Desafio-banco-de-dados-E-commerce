##  Como Executar

1. Crie um novo schema no seu banco de dados (ex.: `ecommerce_dio`);
2. Execute o arquivo **`schema.sql`** para criar as tabelas;
3. Execute o arquivo **`inserts.sql`** para inserir dados de exemplo;
4. Por fim, execute as consultas em **`consultas.sql`** para testar o banco.
